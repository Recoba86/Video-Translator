📋 STEP-BY-STEP INSTRUCTIONS
═══════════════════════════════════════════════════════════════


┌─────────────────────────────────────────────────────────────┐
│  STEP 1: Run the Installation Script                       │
└─────────────────────────────────────────────────────────────┘

Open your terminal and run:

    ./install_mac_arm.sh

⏱️  This takes 5-10 minutes. It will:
   • Install Homebrew (if needed)
   • Install Python 3.11
   • Install FFmpeg & Redis  
   • Create virtual environment
   • Install all packages (ARM-optimized)
   • Test everything

☕ Grab a coffee while it installs!


┌─────────────────────────────────────────────────────────────┐
│  STEP 2: Configure Your API Key                            │
└─────────────────────────────────────────────────────────────┘

The script already created your .env file, but verify:

    cat .env

You should see:

    GEMINI_API_KEY=AIzaSyDErRp1HpEHJVsc5OLM1I88haAAqdIRu5g

✅ If yes, you're good!
❌ If no, run: echo 'GEMINI_API_KEY=AIzaSyDErRp1HpEHJVsc5OLM1I88haAAqdIRu5g' >> .env


┌─────────────────────────────────────────────────────────────┐
│  STEP 3: Start the Services                                │
└─────────────────────────────────────────────────────────────┘

You have TWO options:


🔹 OPTION A: Easy Mode (Recommended)
────────────────────────────────────

Just run:

    ./start.sh

This starts everything in the background!


🔹 OPTION B: Debug Mode (3 Terminals)
──────────────────────────────────────

Terminal 1:  ./start_redis.sh
Terminal 2:  ./start_celery.sh  
Terminal 3:  ./start_flask.sh

Use this if you want to see logs in real-time.


┌─────────────────────────────────────────────────────────────┐
│  STEP 4: Open Your Browser                                 │
└─────────────────────────────────────────────────────────────┘

Go to:

    http://localhost:5000

You should see the beautiful Persian/English UI! 🎉


┌─────────────────────────────────────────────────────────────┐
│  STEP 5: Test It!                                          │
└─────────────────────────────────────────────────────────────┘

1. Paste a video URL (e.g., YouTube link)
2. Click "پردازش ویدیو" (Process Video)
3. Watch the 5-stage progress:
   📥 Downloading
   🎤 Transcribing (Whisper - FREE!)
   🌐 Translating (Gemini - ~$0.01)
   📝 Generating subtitles
   🎬 Burning subtitles
4. Download your video with Persian subtitles!
